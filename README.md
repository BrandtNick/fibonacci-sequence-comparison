## Fibonacci sequence benchmark
Fibonacci sequence benchmark comparison between JS, Go and Rust.

#### Installation
```sh
  git clone https://github.com/BrandtNick/fibonacci-sequence-comparison 
```

#### Usage

Just run the shellscript and

```sh
  ./run.sh
```
